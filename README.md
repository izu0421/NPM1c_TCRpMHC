## TCRdock cancer TCR:pMHC — structures & visualisation

Predicted TCR:peptide–MHC structures (TCRdock / AlphaFold) for a panel of
cancer-associated TCRs, plus the notebooks used to analyse and visualise them.

## Live viewer (GitHub Pages)

`index.html` is a self-contained, interactive 3D viewer showing the three
confirmed binders vs their high-PAE partners coloured by pLDDT, plus the
TCR–peptide / TCR–MHC binding interactions. Enable GitHub Pages (Settings →
Pages → deploy from the default branch, root) and it will be served at the
repo's Pages URL. Regenerate it with `python make_index.py`.

## Contents

```
data/        input tables
  combined_df_merged_stats_pyrosetta_previous_runApr2025.csv   TCR panel + binder labels (query==1) and TCR groups (motif)
  cancer_output_w_pae.tsv                                       TCRdock run output with per-model PAE
structures/  102 predicted TCR:pMHC models (*_model_2_ptm_ft4.pdb)

results/     pre-generated figures/tables (regenerated by the notebooks)
  pmhc_tcr_pae_by_condition.pdf
  binder_binding_interactions.csv
  binding_interactions_methods_results.md
  binder_structures_export/                pLDDT-annotated binder PDBs
```

## Notebooks

| Notebook | What it does |
| --- | --- |
| `pmhc_tcr_pae_by_condition.ipynb` | Box/strip plot of pMHC–TCR PAE by condition (binder / experimental negative / computational negative). |
| `binder_structures_plddt.ipynb` | 3D py3Dmol view of the 3 binders vs their highest-PAE partners, coloured by pLDDT; exports annotated PDBs. |
| `binder_binding_interactions.ipynb` | Detects and renders TCR–peptide / TCR–MHC contacts (salt bridge, H-bond, disulfide, hydrophobic) and writes `binder_binding_interactions.csv`. |

## Running

```bash
pip install -r requirements.txt
jupyter lab   # then run any notebook top-to-bottom
```

All paths in the notebooks are relative to the repository root, so run Jupyter
from this directory.
